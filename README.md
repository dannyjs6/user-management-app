# Тестовое задание

### Модуль управления пользователями

### Запуск проекта

npm install \
npm run start

### Окружение:

Проект разрабатывался и тестировался в окружении:
Node.js: v24.14.1
ОС: WSL (Windows Subsystem for Linux)

### Стэк:

- Vue 3 (Composition API) 
- TypeScript
- Vite
- Pini
- JSON Server (mock API)

---

Проект использует подход с разделением на слои. \
В дальнейшем по мере усложнения можно улучшить до feature-based структуры (features/user/...).

Некоторые вещи были намеренно упрощены из-за ограничений по времени:
- не вынесены таблицы и layout-компоненты (header, layout)
- отсутствует интернационализация (i18n)
- не реализована полноценная feature-based архитектура
- оптимизация структуры store и API
